Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma coleção pessoal de projetos pessoais. O objetivo é criar um ambiente virtual onde seja possível compartilhar seus projetos preferidos, fornecendo uma descrição breve de cada um, incluindo o autor, e o ano de publicação.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo cores, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta o propósito do site e a coleção de projeto em uma seção flexível (.Projetos). Cada projeto é destacado em seu próprio contêiner (.Projetos), mostrando uma imagem dele, detalhes do projeto e um link que leva ao projeto. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis para cores, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de cores suaves e botões interativos. O uso de HTML e CSS é importante pois assim é possível criar um site acessível e esteticamente agradável sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de cores contribui para a atmosfera acolhedora do site, incentivando a exploração da coleção de projetos.
